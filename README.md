# Informe Analitico del Presupuesto General de Honduras
## Autores
- Diego Rosales
- Bryan Arita

## Descripcion
Este proyecto analiza la distribución del gasto público en Honduras utilizando datos abiertos.

## Pregunta de investigación
¿Cuáles son las instituciones que reciben el mayor presupuesto por parte del gobierno de Honduras, según los datos del Portal
de Datos Abiertos de la Secretaría de Finanzas, y determinar si los gastos públicos se realizan de forma equitativa entre las
diferentes instituciones públicas? 

## Dataset
Fuente: Portal de Contrataciones Abiertas de Honduras

## Herramientas
- Python (limpieza de datos)
- Python (visualizacion)
